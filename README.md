# CSI606
## Proposta de Trabalho Final: Tarefinha

**Discente:** Geovana Silva de Oliveira  

---

## Resumo

Este trabalho apresenta a proposta de desenvolvimento do **Tarefinha**, um sistema web para organização e acompanhamento de tarefas domésticas em repúblicas. A ideia surgiu a partir de uma experiência pessoal, em conversas com as moradoras da minha casa, quando percebemos a dificuldade de encontrar um aplicativo ou site que realmente ajudasse na organização das tarefas domésticas de forma clara e justa, sem ser difícil de usar. Assim, o sistema foi pensado para tornar a rotina da república mais organizada e colaborativa.

O sistema busca facilitar essa organização por meio da criação de repúblicas, atribuição de tarefas e acompanhamento do que já foi feito (e do que ainda está esperando alguém criar coragem para fazer).

---

## 1. Tema

O trabalho final tem como tema o desenvolvimento de um **sistema web para organização de tarefas domésticas em repúblicas**, com foco em usabilidade, simplicidade e colaboração entre os moradores.

---

## 2. Escopo

O sistema **Tarefinha** contará com as seguintes funcionalidades:

- Cadastro e autenticação de usuários  
- Criação de repúblicas e ingresso por meio de código de acesso  
- Definição de um líder da república
- Criação e gerenciamento de tarefas domésticas pelo líder  
- Atribuição de responsáveis às tarefas  
- Definição da frequência das tarefas  
- Visualização das tarefas individuais na tela inicial do usuário  
- Visualização das tarefas gerais da casa  
- Marcação de tarefas como concluídas  
- Histórico básico de execução das tarefas  

Como funcionalidades futuras (ainda no campo dos sonhos), pretende-se incluir:
- Calendário de tarefas  
- Filtros avançados  
- Sistema de recompensas ou multas para incentivar a colaboração e evitar a bagunça
- Integração com Firebase

---

## 3. Restrições

Neste trabalho não serão considerados:

- Integração com serviços externos  
- Notificações em tempo real  
- Aplicativo mobile  

O foco do projeto está na definição do escopo, modelagem do sistema e prototipação das funcionalidades principais.

---

## 4. Protótipo

Foram elaborados protótipos para as seguintes páginas do sistema:

- Tela de login  
- Tela de cadastro  
- Tela inicial de boas-vindas (criar ou entrar em república)  
- Home (minhas tarefas)  
- Lista de tarefas da casa  
- Tela de criação de tarefas
